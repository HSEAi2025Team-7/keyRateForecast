И по идее в выходе должны быть графики корреляции.

Интуитивно - если был пресс-релиз о том, что инфляция растет, то потом должен быть пресс-релиз о том, что ставка тоже повысилась, чтобы остановить инфляцию (в нашей разметке из 1/0/-1 это 1)

# Glossary Глоссарий ( полностью от гпт)

### 🔷 1. Источники данных и структура датасета
**Пресс-релизы ЦБ РФ**  Официальные текстовые публикации Банка России, содержащие риторику, которая объясняет решение по ключевой ставке и макроэкономические оценки.

**Ключевая ставка** Основная процентная ставка ЦБ РФ, определяющая стоимость ликвидности в экономике.
Изменение ключевой ставки является целевой переменной проекта.

**Дата заседания** Дата официального заседания совета директоров ЦБ, на котором принимается решение по ставке.

**is_between** Флаг, показывающий, что пресс-релиз относится к периоду между двумя заседаниями.
Используется для обучения модели на релевантных текстах.

**decision** Категориальная метка решения по ставке:
- +1 — ставка повышена
- 0 — сохранена 
- –1 — снижена

**key_rate** Указанный в релизе размер ключевой ставки (например, 15.0%).

### 🔷 2. Лейблы и подготовка данных
**label** Целевая метка, которую прогнозирует модель:
- +1 — на следующем заседании ставка вырастет
- 0 — останется без изменений
- –1 — снизится

**rate_next** Размер ставки на следующем заседании. Используется для расчёта разницы diff.

**diff** Разница между следующей ставкой и текущей: rate_next − rate

### 🔷 3. NLP-предобработка
**clean** Текст после базовой очистки:
- нижний регистр
- удаление переносов
- удаление служебных символов

**sentences** Список предложений, полученных после токенизации по пунктуации.

**tokens** Список слов (токенов):
- без пунктуации
- без цифр
- без стоп-слов
- нормализованных

**stopwords** Частые, малозначимые слова (например: «и», «в», «как», «также», «год»).
Удаляются для улучшения качества признаков.

**lemmas** Лемматизированные слова — приведённые к нормальной форме (морфология через pymorphy3).
Пример: «повышением» → «повышение».

**bigrams** Пары соседних слов (n-grams со значением n=2):
Пример: «инфляционное давление».

**abbr expansion (abb)** Замена сокращений: «б.п.» → «базисный пункт» «млрд» → «миллиард» и т.д.

### 🔷 4. Модели и методы NLP
#### 4.1 TF-IDF
**TF (term frequency)** Частота появления слова в документе.
**IDF (inverse document frequency)** Обратная частота слова в корпусе: чем реже слово, тем выше его вес.
**TF-IDF vectorization** Преобразование текста в числовую матрицу признаков, где каждый признак — вес слова.

#### 4.2 KMeans clustering
**Кластеризация** Объединение релизов в k групп на основе текстового содержания.
**k (число кластеров)** Количество смысловых групп. В проекте: 8.
**centroid (центроид)** Среднее положение кластера в признаковом пространстве.
Показывает, какие слова в кластере самые «важные».

**crosstab** Таблица распределения decision внутри каждого кластера.

#### 4.3 Log-odds Analysis
**log-odds** Логарифм отношения вероятностей встречаемости слова в группе hike к группе cut.
Положительное значение — ассоциация со сохранением/повышением.
Отрицательное — со снижением.
Используется для выявления эмоционально-семантических индикаторов будущих решений ЦБ.

### 4.4 LDA (Latent Dirichlet Allocation)
**LDA** Алгоритм тематического моделирования, выделяющий скрытые темы в текстах.
**topic** Набор слов, образующих семантически связанную тему.
**topic distribution** Вероятность принадлежности документа к каждой теме.
#### 4.5 PMI (Pointwise Mutual Information)
**PMI** Мера ассоциации слов в биграмме.
PMI выше → слова сильнее связаны.

_Пример биграммы с высоким PMI:_
«геополитический риск»
«инфляционное давление»

### 🔷 5. Визуализация
**Stacked bar chart** Показ долей решений (–1/0/+1) по кластерам в виде «стековой» диаграммы.

**Horizontal bar chart (barh)** Используется для:
- log-odds слов
- PMI биграмм
Более удобен для длинных русских слов.

### 🔷 7. Макроэкономические термины
**Инфляционное давление** Набор факторов, увеличивающих темпы роста цен.
**Проинфляционные риски** Угрозы ускорения инфляции в будущем.
**Дезинфляция** Замедление роста цен.
**Проциклическая риторика Риторика ЦБ**, совпадающая с фазой экономического цикла (например, ужесточение политики в условиях перегрева экономики).

### 🔷 8. Общие термины ML
**Feature engineering** оздание признаков на основе текста (TF-IDF, n-grams, bigrams, log-odds, темы).
**Train / Test split** Разделение данных на обучающую и тестовую выборки.
**Classification** Категорийный прогноз решения ЦБ: повышать / снижать / сохранять ставку.
